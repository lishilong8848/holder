公开 API 说明

同步查询接口

外部系统可直接调用此接口。调用时请将下方 URL 中的 127.0.0.1 替换为本机的局域网 IP。

请求路径: http://127.0.0.1:58000/api/v1/query/batch
参数格式:
{
  "people": [
    {"name": "姓名", "id_number": "身份证号"}
  ]
}
CURL 示例:
curl -X POST "/api/v1/query/batch" -H "Content-Type: application/json" -d '{"people":[{"name":"李世龙","id_number":"..."}]}'
